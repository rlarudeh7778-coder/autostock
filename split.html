⚙️ 최초 1회 설정이 필요해요 (5분)
주식 시세는 프록시 서버가 있어야 받아져요. 전에 네이버 API 때 하신 것과 똑같아요:
1) dash.cloudflare.com → Workers & Pages → Create Worker
2) 같이 드린 worker.js 코드 붙여넣고 Deploy
3) 나온 주소(https://xxx.workers.dev)를 이 파일 코드 맨 위 WORKER_URL에 넣고 다시 업로드
시세를 불러오지 못했어요. WORKER_URL 설정과 인터넷 연결을 확인해주세요.
세븐스플릿이 뭐예요? 사용법 (1분)
원리: 계좌를 번호로 나눠요. 1번은 장기투자자(+30%까지 안 팔고 보유), 2번부터는 단타부대 — 주가가 정한 간격만큼 떨어질 때마다 다음 번호가 사고, 각 번호는 자기 매수가에서 +5% 오르면 자기만 팔아요. 평단으로 뭉치지 않으니까 "물렸다"는 개념이 사라지고, 하락장에서도 밑 번호들이 사고팔며 수익을 만들어요.
1발굴 목록에서 종목 선택. 52주 가격 범위에서 아래쪽에 있을수록(할인 클수록) 스플릿 시작하기 좋아요. 여긴 애초에 망할 걱정 적은 시총 상위 대형주만 모아뒀어요.
2플랜 만들기. 총 투자금, 몇 번호로 나눌지(추천 5), 매수 간격(추천 -5%), 익절(추천 +5%)을 넣으면 번호별 매수 가격표가 자동으로 나와요. 1번만 지금 사고 시작!
3매일 1번 열기. "N번 매수 시점" 뜨면 그 번호 금액만큼 매수 → 체결가 입력하고 기록. "N번 매도 시점" 뜨면 그 번호 수량만 매도 → 기록. 끝. 시스템이 시키는 것만 하면 분할매수·분할매도가 자동으로 지켜져요.
⚠️ 3원칙 (박성현식): ① 손절 없음 — 대신 망하지 않을 우량주만, 여유자금만 ② 레버리지·신용·미수 절대 금지 ③ 정한 간격·금액 임의로 바꾸지 않기. 손절이 없는 방식이라 하락이 길어지면 돈이 오래 묶일 수 있어요. 반드시 여유자금으로만 하세요.
